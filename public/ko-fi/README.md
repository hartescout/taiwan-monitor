# Ko-fi Assets

This folder holds copy and asset guidance for the Pharos Ko-fi page.

Purpose:
- keep donation messaging consistent with the project mission
- prepare profile copy before the live Ko-fi page is finalized
- document what visuals should be used for the Ko-fi profile

Files:
- `intro.md` - short page intro options
- `bio.md` - short and medium bio options
- `profile-icon.md` - profile image recommendation
- `cover-image.md` - cover image recommendation
- `logo.svg` - copied source logo asset
- `logo.png` - PNG export for Ko-fi profile usage
- `logo-mark.svg` - square P-only logo mark for avatar use
- `logo-mark.png` - PNG export of the square P-only logo mark
- `cover-base.png` - copied dashboard screenshot for cover image usage

Funding policy:
- Pharos is not being positioned as a profit project
- support is meant to cover AI usage, hosting, database, map, and infrastructure costs
- the intent is to keep the dashboard online and improving, not to extract profit

Current suggested source assets:
- logo base: `public/ko-fi/logo.svg`
- logo PNG: `public/ko-fi/logo.png`
- avatar mark: `public/ko-fi/logo-mark.png`
- cover base: `public/ko-fi/cover-base.png`
