# Profile Icon Guidance

## Recommended Direction

Use the square P-only Pharos mark as the profile image.

Base assets:
- `public/ko-fi/logo-mark.svg`
- `public/ko-fi/logo-mark.png`

## Recommendation

- use the square dark-background mark with the single `P`
- keep generous padding around the wordmark
- avoid adding extra text like the full bio or URL
- prioritize clarity at very small sizes

## Visual Notes

- background should stay dark and consistent with the product identity
- the `P` mark should remain the focal point
- avoid thin decorative details that disappear in small avatars

## Suggested Deliverable

- square PNG export for Ko-fi profile avatar
- dark background
- centered Pharos wordmark

Current ready-to-use file:
- `public/ko-fi/logo-mark.png`
