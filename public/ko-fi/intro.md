# Ko-fi Intro

## Recommended

Pharos is an open-source conflict intelligence dashboard built to make fast-moving conflicts easier to understand. Instead of isolated headlines, it brings together events, actor responses, map stories, signals, and daily briefs into one live system.

If the project has been useful to you, support helps keep it online and helps cover the infrastructure required to run it.

## Shorter Option

Pharos turns fragmented conflict reporting into a live, structured intelligence dashboard. Support helps keep the project online and sustainable.

## Warmer Option

Pharos exists to help people understand conflicts as systems, not just scattered updates. If you want to help keep it running, your support goes directly toward the infrastructure behind the project.
